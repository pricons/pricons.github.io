Installation Instructions

Firefox

This extension is downloadable for Firefox right here.

Chrome/Brave/Edge (Difficult)

The extension is currently not available from the Chrome Web Store, due to a 5 dollar fee, and I don't wanna set up a credit card. Also my parents wouldn't support a pride extension. :/

Go to the url chrome://extensions, and then install this, and extract the zip file. Go to the tab again, and enable developer mode. Click load unpacked extension. Now upload the contents of the zip file.
